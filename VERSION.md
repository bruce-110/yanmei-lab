# 版本信息

## 版本号：v2025.01.03

## 发布日期：2025年1月3日

## 版本说明

这是颜美实验室（YANMEI LAB）的初始版本。

## 主要功能

1. **AI审美评分**
   - 使用通义千问 VL Plus 模型
   - 100分满分评分系统
   - 视觉年龄评估

2. **审美点评**
   - 犀利的毒舌点评
   - 穿搭问题分析
   - 改进建议

3. **长图生成**
   - 一键生成长图
   - 极简现代设计风格
   - 莫兰迪配色方案

4. **多语言支持**
   - 中文（简体）
   - 英文

5. **使用次数管理**
   - 免费用户10次限制
   - 数据持久化存储

## 技术栈

- Python 3.10+
- Streamlit
- 通义千问 API (DashScope)
- Pillow (图像处理)
- python-dotenv (环境变量管理)

## 配色方案

- 背景色：#F5F2F0（浅米白）
- 强调色：#8B4B5C（深酒红色）
- 文字色：#333333（深灰色）
- 辅助色：#E8E4E1（浅灰色）

## 部署平台

- Replit（推荐）
- 本地运行
- 其他支持Streamlit的平台

## 环境变量

- `DASHSCOPE_API_KEY`: 通义千问API密钥
